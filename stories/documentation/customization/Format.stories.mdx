# Format

Some components have optional props `format` for date and time display.

## Default format

| Type | Format                 |                                                               |
| ---- | ---------------------- | ------------------------------------------------------------- |
| Date | Base on `locale` props | `locale.formatLong?.date({ width: 'short' }) ?? 'MM/dd/yyyy'` |
| Time | `HH:mm`                |                                                               |
