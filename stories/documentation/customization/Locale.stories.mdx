# Locale

All components need a `locale` props for translation and locale base display.

```tsx
import React, { FC } from 'react';

import { fr } from 'date-fns/locale';

import { Calendar } from 'react-next-dates';

const MyComponent: FC = () => {
  return <Calendar locale={fr} />;
};
```
