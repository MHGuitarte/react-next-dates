## Using Sass

Sass default theme :

```scss
$react-next-dates-theme: map-merge(
  (
    color-primary: #2979ff,
    color-grey: #6b6d77,
    color-light-grey: #f2f2f2,
    color-dark-grey: #2c2e3d,
    color-white: #ffffff,
    color-dark: #000000,
    font-size-small: 12px,
    font-size-base: 14px,
    font-size-large: 16px,
    font-size-extra-large: 18px,
  ),
  $react-next-dates-theme
);
```

You can override the theme with :

```scss
$react-next-dates-theme: (
  color-primary: #06c9b2
);

@import 'react-next-dates/dist/style.scss';
```

## Using CSS

React next dates use css variables for css theming.

If you use css or css in js library like style-components you can override css variables on you code.

**All css variables :**

```css
.react-next-dates {
  --color-primary: #2979ff;
  --color-grey: #6b6d77;
  --color-light-grey: #f2f2f2;
  --color-dark-grey: #2c2e3d;
  --color-white: #ffffff;
  --color-dark: #000000;
  --font-size-small: 12px;
  --font-size-base: 14px;
  --font-size-large: 16px;
  --font-size-extra-large: 18px;
}
```

**In your code you can override variables :**

```css
.react-next-dates {
  --color-primary: #06c9b2;
}
```

> If your browser don't support [CSS Variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties#Browser_compatibility), library use sass theme.

All components have optional props `className` to inject your custom class or to style them with css in js library.

**Example :** [calendar story](?path=/docs/components-calendar--default-story) inject `story-calendar` className to add border.
